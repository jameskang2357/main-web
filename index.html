About Me

I'm an Infrastructure Engineer at Valon, where I build and run the infrastructure behind our data platform: multi-tenant data architecture, infrastructure as code with Terraform, and observability across roughly 100 Airflow DAGs. I also built the infrastructure for an internal LLM-powered agent that handles thousands of Slack invocations a day.

Before Valon, I spent four years at BILL as a Data Engineer owning batch and streaming pipelines built on AWS Kinesis, Kafka, and Flink. Before that I was a Software Developer at Avant, shipping full-stack features and putting machine learning models into production. I enjoy the intersection of infrastructure, data engineering, and solving real-world problems at scale.

When I'm not building pipelines, I'm learning about new technologies, writing about my experiences, and contributing to open-source projects.
